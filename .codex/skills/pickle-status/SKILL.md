# Pickle Rick Status

Run: `node $HOME/.codex/pickle-rick/bin/status.js`

Display current session state including:
- Active/inactive flag
- Current step and ticket
- Iteration count / max
- Circuit breaker state
- Time elapsed
- Session identifier
