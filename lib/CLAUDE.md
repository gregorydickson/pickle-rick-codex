## Trap Doors

- `detached-launch.js` — INVARIANT: empty fresh launch/cwd locks keep the write grace; non-empty non-PID payloads are corrupt and reclaimable. BREAKS: relaunches either steal live reservations or wedge behind junk lockfiles. ENFORCE: `advanced loop resume refuses to relaunch while a fresh empty tmux launch lock is held`; `pickle-tmux clears a fresh malformed launch lock before relaunching`. PATTERN_SHAPE: `fs.openSync(lockPath,'wx')` EEXIST handlers that apply the empty-file grace to any unreadable payload.
- `verification-env.js` — INVARIANT: scoped package-manager test rewrites must honor package cwd flags before reading `package.json`. BREAKS: `npm|pnpm|yarn|bun test -- <file>` falls back to the broad wrapper and runs the wrong suite. ENFORCE: `spawn-morty rewrites npm --prefix scoped vitest verification commands into targeted execution`. PATTERN_SHAPE: `extractPackageDirFromTokens()` parsers that handle `test -- <file>` but omit manager-specific cwd selectors such as `--prefix`.
